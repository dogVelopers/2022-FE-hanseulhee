## HTTP Method

<b>HTTP Method란?</b>

클라이언트와 서버 사이에 이루어지는 요청(request)과 응답(response) 데이터를 전송하는 방법

GET : 리소스 조회 (가져오기)

POST : 요청 데이터 처리, 주로 데이터 등록에 사용

PUT : 리소스를 대체, 해당 리소스가 없으면 생성

DELETE : 리소스 삭제
